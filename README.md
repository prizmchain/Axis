# Axis

**Axis** is a macOS screen dimming app that lets you control the brightness of each monitor independently.

## Features

- Per-monitor brightness control with intuitive overlay dimming
- Multi-monitor support with screen selector
- Quick presets for common brightness levels
- Fine-tuning with arrow keys
- Keyboard shortcut support (ESC to close)
- VoiceOver accessibility
- Lightweight and runs in the menu bar

## System Requirements

- macOS 13.0 (Ventura) or later

## Installation

1. Download the latest `.dmg` from [Releases](https://github.com/prizmchain/Axis/releases)
2. Open the DMG and drag **Axis** to your Applications folder
3. Launch Axis from your Applications folder
