# Axis

**Axis** is a macOS menu bar app that lets you control the position of notification banners on your screen. Move notifications to any corner, edge, or center of your display.

macOS always shows notification banners in the top-right corner. Axis changes that -- place them wherever you want.

---

## Features

- Move macOS notification banners to any position on screen
- Multi-monitor support with screen selector
- Quick position presets (Top Right, Top Center, Center, Bottom Right)
- Fine-tune position with arrow keys
- Lightweight menu bar app (no Dock icon)
- Launch at Login support
- VoiceOver accessibility labels and adjustable actions
- 3-step onboarding for first-time users

## System Requirements

- macOS 14.0 (Sonoma) or later
- Accessibility permission (required to reposition notification windows)

## Installation

1. Download the latest `.dmg` from [Releases](https://github.com/prizmchain/Axis/releases)
2. Open the DMG and drag **Axis** to your Applications folder
3. Launch Axis from your Applications folder
4. When prompted, grant Accessibility permission in System Settings

## License Activation

Axis requires a license key to use.

1. On first launch, the license activation window will appear
2. Enter your license key and click **Activate**
3. Each license key is tied to a single device
4. After activation, grant Accessibility permission when prompted

You can purchase a license key from [our website](https://github.com/prizmchain/Axis).

## Usage

1. Click the **Axis** icon in the menu bar
2. Select **Set Position...** to open the position picker
3. Drag the crosshair or use arrow keys to fine-tune placement
4. Use **Quick Positions** for common presets
5. Notifications will move to your chosen position automatically

# Axis (한국어)

**Axis**는 macOS 알림 배너의 위치를 자유롭게 조절할 수 있는 메뉴 바 앱입니다.

macOS는 항상 알림을 화면 오른쪽 상단에 표시합니다. Axis를 사용하면 원하는 위치 어디든 알림을 배치할 수 있습니다.

## 주요 기능

- macOS 알림 배너를 화면 어디든 이동
- 멀티 모니터 지원 (화면 선택 가능)
- 빠른 위치 프리셋 (우측 상단, 중앙 상단, 중앙, 우측 하단)
- 화살표 키로 미세 조정
- 가벼운 메뉴 바 앱 (Dock 아이콘 없음)
- 로그인 시 자동 실행
- VoiceOver 접근성 지원

## 시스템 요구사항

- macOS 14.0 (Sonoma) 이상
- 손쉬운 사용(접근성) 권한 필요

## 설치 방법

1. [Releases](https://github.com/prizmchain/Axis/releases) 페이지에서 최신 `.dmg` 파일 다운로드
2. DMG를 열고 **Axis**를 응용 프로그램 폴더로 드래그
3. 응용 프로그램 폴더에서 Axis 실행
4. 안내에 따라 시스템 설정에서 손쉬운 사용 권한 허용

## 라이선스 활성화

Axis는 라이선스 키가 필요합니다.

1. 첫 실행 시 라이선스 활성화 창이 표시됩니다
2. 라이선스 키를 입력하고 **Activate**를 클릭하세요
3. 각 라이선스 키는 하나의 기기에만 사용 가능합니다
4. 활성화 후 손쉬운 사용 권한을 허용해 주세요
